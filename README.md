Legacy WinForms Management System – VB.NET

Sistema de gestión comercial desarrollado en WinForms con VB.NET, utilizado como ejemplo real de mantenimiento y evolución de aplicaciones legadas productivas en entornos corporativos.

Este proyecto refleja escenarios comunes en empresas grandes, bancos y organizaciones donde el negocio depende de software construido hace muchos años y que debe seguir funcionando sin margen de error.
